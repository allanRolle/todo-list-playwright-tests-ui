# Projet: [To do application](https://angular-todo-app-with-signals.netlify.app/)

## Objectif:

Réaliser des tests end-to-end pour tester toutes les règles métier de l'application:

- Ajouter un todo
- Editer un todo
- Supprimer un todo
- Sélectionner tous les todos
- Sélectionner tous les todos et dé-sélectionner un
- Filter la liste de todos
  - afficher tous les todos
  - afficher les todos en cours
  - afficher les todos terminés
- Supprimer tous les todos terminés

## Méthodologies

- BDD / Gherkin
- POM (Page Object Model)

## Environnement technique

- Playwright
- Cucumber
- Github actions (CI/CD)

## Installation

- Cloner le répertoire
- Installer les dépendences avec `npm install`
- Lancer les tests en mode non interactif avec la commande `npm run test`
